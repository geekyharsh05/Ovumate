# OvuMate - Your Cycle, Your Control

It is a simple Ovulation Calculator web app built using Next.js, TypeScript, Tailwind CSS.

## Requirements

- Docker

or

- node
- npm(preferred)

## Installation

### With docker

<details>
<summary>
Click Me
</summary>

1. Clone the repository
2. Use docker-compose to run the server

```bash
git clone git@github.com:geekyharsh05/Ovumate.git
docker-compose up -d
```

</details>

### Without docker

<details>
<summary>
Click Me
</summary>

1. Clone the repository
2. Install the dependencies
3. Run the server

```bash
git clone git@github.com:geekyharsh05/Ovumate.git
cd Ovumate
npm install
npm run dev
```

</details>
